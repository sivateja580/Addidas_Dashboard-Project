# Adidas Dashboard Project (Power BI)
This is a Power BI project I created to analyze Adidas sales data.  
The dashboard helps in understanding sales performance, revenue trends, and product analysis.
## Files in this repository
- Adidas_DashBoard.pbix – The main Power BI dashboard file.  
- dashboard.png – A screenshot of the dashboard for a quick preview.  
## Dashboard Preview
![Dashboard Screenshot](dashboard.png)
## How to use this project
1. Download the file `Adidas_DashBoard.pbix` from this repository.  
2. Open the file using Power BI Desktop.  
3. Explore the dashboard and interact with the different visuals.  
## What this dashboard shows
- Sales revenue trends across different regions.  
- Best-selling products and categories.  
- Comparison of sales performance over time.  
- Insights that can help in business decision-making.  
---
This project was created to practice Power BI and to learn how to build dashboards for business insights.
